# Noa-slide-puzzle
ブルーアーカイブのノアのイラストを使用したスライドパズルです

自分で遊ぶために作ったため、ブルーアーカイブのBGMをそのまま使用しました

そのためゲーム自体はアップロードできません、申し訳ないです

# 遊び方
1.スタートを押したらゲーム画面に移行します

2.隣に空きがあるピースをクリックすると空いてる方向へピースが移動します

3.スライドさせていくことで絵を完成させましょう!
